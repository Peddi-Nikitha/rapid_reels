# Rapid Reels

Instant Event Reel Platform - A Flutter application for creating and delivering instant event reels.

## Project Overview

Rapid Reels is a mobile application that enables users to book instant event reel services for various occasions including weddings, birthdays, engagements, corporate events, and brand promotions.

## Repository Structure

- `rapid_reels_app/` - Main Flutter application
- `files/` - Project documentation and guides
